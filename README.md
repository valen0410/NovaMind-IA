# NovaMind-IA
Proyecto de IA hecho por Valentina Fernandez Alegre

NovaMind AI es una plataforma web profesional desarrollada con HTML, CSS y JavaScript.

La página permite:
- recibir consultas mediante un chatbot interactivo
- responder automáticamente mensajes del usuario
- enviar información a través de un formulario de contacto
- mostrar una interfaz moderna y responsive

El proyecto fue diseñado como una aplicación web enfocada en tecnologías modernas e integración de asistentes virtuales.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Chat interactivo
- Respuestas automáticas
- Formulario de contacto
- Diseño responsive
- Interfaz profesional

## Objetivo

Desarrollar una plataforma moderna de consultas inteligentes orientada a proyectos web e integración de IA.
